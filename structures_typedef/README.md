# Structures and Typedef

This directory contains C programs that use structures and typedef.

## Files

- `0-main.c`: Contains the main function for the `0-init_dog.c` program.
- `1-init_dog.c`: Contains a function that initializes a variable of type `struct dog`.
- `1-main.c`: Contains the main function for the `1-init_dog.c` program.
- `2-print_dog.c`: Contains a function that prints a `struct dog`.
- `2-main.c`: Contains the main function for the `2-print_dog.c` program.
- `3-main.c`: Contains the main function for the `3-prototype.c` program.
- `4-new_dog.c`: Contains a function that creates a new dog.
- `4-main.c`: Contains the main function for the `4-new_dog.c` program.
- `5-free_dog.c`: Contains a function that frees dogs.
- `5-main.c`: Contains the main function for the `5-free_dog.c` program.
- `dog.h`: Contains header file declarations for the programs.
- `README.md`: Contains information about the programs.

## Usage

To use these functions in your own program, include the `dog.h` header file and call the corresponding function in your code.

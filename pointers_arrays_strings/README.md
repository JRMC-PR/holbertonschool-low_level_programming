# Pointers, Arrays, and Strings

This directory contains C programs that use pointers, arrays, and strings.

## Files

- `0-memset.c`: Contains a function that fills memory with a constant byte.
- `0-reset_to_98.c`: Contains a function that updates the value of an integer to 98.
- `0-strcat.c`: Contains a function that concatenates two strings.
- `1-swap.c`: Contains a function that swaps the values of two integers.
- `1-memcpy.c`: Contains a function that copies memory area.
- `1-strncat.c`: Contains a function that concatenates two strings, using at most n bytes from src.
- `2-strlen.c`: Contains a function that returns the length of a string.
- `2-strchr.c`: Contains a function that locates a character in a string.
- `2-strncpy.c`: Contains a function that copies a string.
- `3-main.c`: Contains the main function for the `3-puts.c` program.
- `3-puts.c`: Contains a function that prints a string, followed by a new line, to stdout.
- `3-strcmp.c`: Contains a function that compares two strings.
- `3-strspn.c`: Contains a function that gets the length of a prefix substring.
- `4-main.c`: Contains the main function for the `4-print_rev.c` program.
- `4-print_rev.c`: Contains a function that prints a string, in reverse, followed by a new line.
- `4-rev_array.c`: Contains a function that reverses the content of an array of integers.
- `4-strpbrk.c`: Contains a function that searches a string for any of a set of bytes.
- `5-main.c`: Contains the main function for the `5-rev_string.c` program.
- `5-rev_string.c`: Contains a function that reverses a string.
- `5-string_toupper.c`: Contains a function that changes all lowercase letters of a string to uppercase.
- `5-strstr.c`: Contains a function that locates a substring.
- `6-cap_string.c`: Contains a function that capitalizes all words of a string.
- `6-cap_string_variant.c`: Contains a function that capitalizes the first letter of each word and converts all other letters to lowercase.
- `6-main.c`: Contains the main function for the `6-puts2.c` program.
- `6-puts2.c`: Contains a function that prints every other character of a string, starting with the first character, followed by a new line.
- `7-1337.c`: Contains a function that encodes a string into 1337.
- `7-main.c`: Contains the main function for the `7-print_chessboard.c` program.
- `7-print_chessboard.c`: Contains a function that prints the chessboard.
- `7-leet.c`: Contains a function that encodes a string into 1337.
- `7-main.c`: Contains the main function for the `7-print_chessboard.c` program.
- `7-print_chessboard.c`: Contains a function that prints the chessboard.
- `7-puts_half.c`: Contains a function that prints half of a string.
- `8-main.c`: Contains the main function for the `8-print_array.c` program.
- `8-print_array.c`: Contains a function that prints n elements of an array of integers.
- `9-main.c`: Contains the main function for the `9-strcpy.c` program.
- `9-strcpy.c`: Contains a function that copies the string pointed to by src, including the terminating null byte, to the buffer pointed to by dest.
- `100-atoi.c`: Contains a function that converts a string to an integer.
- `main.h`: Contains header file declarations for the programs.
- `README.md`: Contains information about the programs.

## Usage

To use these functions in your own program, include the `main.h` header file and compile your program with the corresponding `.c` files.
